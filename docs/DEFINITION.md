# Definition: Ask Kamos

## Core Concept
**"The Mirror of Thought" (思考の鏡)**

Ask Kamosは、ユーザーの問いに即座に答えるだけのツールではありません。
ユーザーの言葉を鏡のように映し出し、異なる角度から光を当てることで、ユーザー自身も気づいていなかった「思考の死角」や「新しい可能性」を照らし出すパートナーです。

## Key Personas
- **The Director**: 映画監督のように全体を俯瞰し、「カット！もう少しこの観点で深掘りしてみよう」と指示を出すメタ認知エージェント。
- **The Analyst**: 論理的に情報を整理し、構造化する実務家。
- **The Creative**: 飛躍したアイデアやビジュアルイメージを提案する右脳派。

## System Architecture (Agentic)
Ask Kamosは **"Thinking Loop"** と呼ばれる独自の推論ループを持ちます。
1. **Input**: ユーザーの曖昧な問い。
2. **Decomposition**: 問いを複数のサブタスクに分解。
3. **Execution**: 各ペルソナが必要なツール（Google検索、ナレッジベース参照）を使用して分析。
4. **Synthesis**: 結果を統合し、新たな「問い」を生成してループする（Directorの判断による）。
5. **Output**: 構造化されたインサイトと、次のアクションプラン。
