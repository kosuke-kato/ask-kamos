# Ask Kamos

Ask Kamos は、Google Gemini と Kamos Framework Analysisがマトリクスを材料にした対話を通じて、入力された問いを多角的に深掘りするプレイグラウンド（遊び場）です。

## 概要
ユーザーの入力を受けて、AIが自律的に3段階の深掘り指示を生成し、それぞれのフェーズで分析を重ねます。最終的に、これまでのすべてのプロセスを統合したまとめを提供します。

## 機能
- **3-Phase Analysis**: 初期分析、深掘り、そして統合。
- **Gemini 3 Integration**: 最新の Gemini 3 Flash Previewによる高速な思考プロセスの生成。
- **Streaming UI**: 分析の進捗をリアルタイムに確認できるストリーミングインターフェース。
